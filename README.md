# My Portfolio

Welcome to my Portfolio Website! 🌐 Here, I've carefully curated a collection of my best work, showcasing my expertise in web development, design, and coding. This portfolio is a testament to my passion for creating visually stunning and technically sound websites.

## Development Setup

> **Note:** This project requires **Node.js >= 24.0.0**. If you use `nvm`, simply run `nvm use` in the root directory to pick up the correct version from `.nvmrc`.

The project is built using modern tooling including **Vite**, **ESLint**, and **Prettier** to ensure it is deployment-ready.

### 1. Install Dependencies
```bash
npm install
```

### 2. Start Local Server
```bash
npm run dev
```

### 3. Lint and Format Code
```bash
npm run lint
npm run format
```

### 4. Build for Production
```bash
npm run build
```
